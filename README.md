# 野鳥鳴き声サーチ Android 配布

iOS 版「野鳥鳴き声サーチ」の **Android 版 APK 配布専用** リポジトリです。

Play Store 経由ではなく、本リポジトリの GitHub Releases から直接 APK をダウンロードしてインストールしていただく形式です。ソースコードは別リポジトリで管理しており、こちらにはビルド済みの APK のみ置いています。

---

## ダウンロード

👉 [**最新版の Releases ページ**](https://github.com/wat--/releases/releases/latest)

`birdcompass-android-vX.Y.Z.apk` という名前のファイルをタップしてダウンロードしてください。

> ⚠️ **Android 端末で直接アクセスしてください**。iPhone でこのページを開いても APK はインストールできません。

---

## インストール手順 (詳細)

### ステップ 1: APK をダウンロード

1. **Android 端末の Chrome (または好みのブラウザ)** で [Releases ページ](https://github.com/wat--/releases/releases/latest) を開く
2. `birdcompass-android-vX.Y.Z.apk` のリンクをタップ
3. 「このファイルは端末や個人情報に危害を加える可能性があります。続行しますか?」と聞かれたら **「OK」** をタップ
   - これは Play Store 外配信アプリへの一般的な警告です。本アプリは安全ですので「OK」で進めて大丈夫です。
4. ダウンロード完了 (約 130MB、回線によって 30秒〜数分)

### ステップ 2: APK を開く

ダウンロード完了後の選択肢:

- **方法 A** (簡単): 画面上部の通知バーを引き下げて、ダウンロード完了の通知をタップ
- **方法 B**: 「ファイル」アプリ → 「ダウンロード」フォルダ → `birdcompass-android-vX.Y.Z.apk` をタップ

### ステップ 3: 「不明なアプリのインストール」を許可

初めて Play Store 外の APK をインストールする時、次のような画面が出ます:

> 🚫 **このソースからアプリをインストールすることは許可されていません**
>
> 不明なソース [Chrome 等] からのアプリのインストール

設定への遷移ボタンをタップして:

1. 「**この提供元のアプリを許可**」のスイッチを **ON** に切り替え
2. 戻る (←) ボタンで前の画面に戻る

これで Chrome (など、APK をダウンロードしたアプリ) から APK インストールができるようになります。

### ステップ 4: インストール

1. 「このアプリをインストールしますか?」の画面で **「インストール」** をタップ
2. インストール中... (10〜30 秒)
3. **「Play プロテクト」** が「未確認のアプリです」のような警告を出す場合があります → **「とにかくインストール」** または **「詳細」→ 「インストール」** で進める
4. 「完了」をタップ

### ステップ 5: 起動

1. アプリ一覧 (ホーム画面下からスワイプアップ) で **「野鳥鳴き声サーチ」** を探してタップ
2. 初回起動時にマイク権限のダイアログが出るので **「アプリの使用時のみ」** をタップして許可

## 動作環境

| 項目 | 要件 |
|---|---|
| OS バージョン | **Android 8.0 (API 26) 以上** |
| 推奨 | Android 10 以上 / RAM 4GB 以上 |
| 必要権限 | マイク (必須) / 位置情報 (任意) / メディア (写真添付用、任意) |
| ストレージ容量 | 約 250MB (アプリ本体 + モデルファイル + キャッシュ) |
| インターネット接続 | 図鑑の Wikipedia / iNaturalist 画像取得・解説に必要 (オフラインでも基本機能は動作) |

### 動作確認済み端末

- Google Pixel 5 (Android 14)

## トラブルシューティング

### Q. APK タップしても何も起きない

- ダウンロードが完了していない可能性。「ファイル」アプリ → 「ダウンロード」フォルダで確認
- ファイル名が `*.apk` で終わっていることを確認

### Q. 「インストール中にエラーが発生しました」

- ストレージ空き容量が 300MB 以上あることを確認
- 既存版がある場合、一度アンインストールしてから再インストール

### Q. 「不明なソース」の設定画面が出てこない

- Android 7 以下: 設定 → セキュリティ → 「提供元不明のアプリ」を ON
- Android 8 以上: APK を開く時に自動でそのアプリ用の許可設定画面に飛びます

### Q. 起動するとすぐ落ちる

- Android バージョンが 8.0 未満だと動きません。設定 → 端末情報 で確認してください
- ストレージ空き容量を確認
- 一度アンインストールして再インストール

### Q. マイク権限を間違えて拒否してしまった

- 設定 → アプリ → 野鳥鳴き声サーチ → 権限 → マイク → 「アプリの使用時のみ許可」 に変更

### Q. その他の不具合

[Issues](https://github.com/wat--/releases/issues) でご報告ください。
