# 野鳥鳴き声サーチ Android 配布

iOS 版「野鳥鳴き声サーチ」の **Android 版 APK 配布専用** リポジトリです。

ソースコードは別リポジトリで管理しており、こちらにはビルド済みの APK のみ置いています。

## ダウンロード

[**Releases**](https://github.com/wat--/releases/releases) から最新の APK をダウンロードしてください。

## インストール手順

1. Android 端末で Releases ページの APK リンクをタップしてダウンロード
2. 通知から APK ファイルを開く
3. 「不明なアプリのインストールを許可」のダイアログが出たら「許可」
4. 「野鳥鳴き声サーチ」がインストールされます

## 対応環境

- Android 8.0 (API 26) 以上
- マイクと位置情報のパーミッションを使用
- インターネット接続 (図鑑の Wikipedia / iNaturalist 画像取得のため)
